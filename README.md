# skylar-helper
helper for skylar developers

```
skylar-helper Commands [Options]

Commands:
  new-page <PageName>  创建一个页面                            [aliases: n, new]

Options:
  -j, --script-extension  定义JS文件后缀名                       [default: "js"]
  -c, --style-extension   定义样式文件后缀名                   [default: "less"]
  -h, --help              Show help                                    [boolean]
```
